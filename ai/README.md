# AI Skills and Workflows

这里用于保存个人常用的 AI skill、workflow、模板和零散笔记。

## 目录

- `skills/`：可复用的个人技能说明，例如某类任务的固定处理方法。本机 Codex skills 备份在 `skills/codex-local/`，清单见 `skills/MANIFEST.md`。
- `workflows/`：跨多步任务的流程文档，例如每日启动、代码审查、资料整理等。
- `templates/`：prompt、文档结构、检查清单等可复制模板。
- `notes/`：还没有沉淀成 skill 或 workflow 的想法、草稿和实验记录。

## 建议格式

- skill 使用 `SKILL.md` 或 `README.md` 作为入口。
- workflow 使用动词开头的短文件名，例如 `daily-startup.md`。
- 模板文件尽量保持可直接复制使用。
